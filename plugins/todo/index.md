# Todo - 任务管理插件

![Todo Plugin](/icon.png)

✅ **Todo 任务管理** - 高效完成任务，成就更多可能

## 核心功能

### 📝 任务创建与管理
- **快速添加**：一键创建新任务
- **自然语言**：支持"明天3点开会"等自然语言输入
- **任务分组**：按项目、类别、优先级分组
- **任务模板**：使用模板快速创建常用任务
- **批量操作**：批量标记、删除、移动任务

### 📅 日期与时间
- **截止日期**：为任务设置明确的截止时间
- **开始日期**：推迟任务开始时间
- **全天任务**：全天类型的任务支持
- **重复任务**：设置每日、每周、每月重复
- **时间预估**：记录任务预计耗时

### 📊 优先级管理
- **四级优先级**：紧急重要、重要、紧急、不重要
- **颜色标识**：不同优先级使用不同颜色
- **自动排序**：按优先级自动排序任务
- **优先级筛选**：快速筛选特定优先级任务
- **进度追踪**：记录任务完成进度

### 🎯 子任务支持
- **无限层级**：支持创建无限层级的子任务
- **进度汇总**：子任务完成度自动汇总到父任务
- **独立管理**：每个子任务可单独设置截止日期
- **折叠展开**：方便浏览大型任务结构

## 操作指南

### 创建任务
1. 点击「+」按钮创建新任务
2. 输入任务标题
3. 设置截止日期（可选）
4. 选择优先级
5. 添加详细描述
6. 设置提醒时间
7. 保存任务

### 管理子任务
1. 在任务下点击「添加子任务」
2. 输入子任务内容
3. 设置子任务属性
4. 拖拽调整任务层级
5. 勾选完成子任务

### 筛选任务
1. 使用顶部筛选器
2. 选择截止日期范围
3. 按优先级筛选
4. 按状态筛选（未开始/进行中/已完成）
5. 使用搜索框搜索任务

## 常见问题

**Q: 如何设置重复任务？**
A: 在任务详情中选择「重复」，设置重复周期。

**Q: 可以导出任务列表吗？**
A: 可以，导出为 CSV 或 PDF 格式。

**Q: 如何批量编辑任务？**
A: 长按任务进入多选模式，批量操作。

**Q: 任务有数量限制吗？**
A: 没有硬性限制，但建议保持合理的任务数量。
