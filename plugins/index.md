# 插件中心

## 插件开发

Memento 支持自定义插件开发，您可以：

1. **启用/禁用**：根据需要启用或禁用特定插件
2. **自定义配置**：调整插件参数以满足个人需求
3. **数据导入导出**：备份和恢复插件数据
4. **开发新插件**：使用插件开发工具创建专属功能

## 使用建议

- **新手推荐**：从 Chat、Diary、Notes 三个核心插件开始
- **效率提升**：配合 Activity、Todo、Bill、Timer 使用
- **生活管理**：结合 Goods、Checkin、Habits、Contact 管理日常
- **深度用户**：探索 Database、Tracker、Calendar、Store 等高级功能
