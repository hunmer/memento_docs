# Contact - 联系人插件

<PluginSwiper plugin-name="contact" />

👥 **Contact 联系人管理** - 管理人际网络，珍惜每一份关系

联系人插件是 Memento 的人际关系管理模块，提供完整的联系人信息管理、交互记录跟踪和智能筛选排序功能。支持自定义头像、标签系统和灵活的字段定义，帮助您维护重要的人际关系。

## 核心功能

### 📇 联系人管理
- **基本信息**：姓名（必填）、电话号码、地址、备注
- **头像与图标**：支持自定义头像上传和图标选择，带有主题色背景
- **标签系统**：为联系人添加多个自定义标签（支持颜色区分）
- **自定义字段**：灵活的键值对结构，存储个性化信息
- **视图模式**：支持列表视图和网格视图切换显示

### 📞 交互记录
- **互动记录**：记录每次与联系人的互动（时间、内容、参与者）
- **多种类型**：支持电话、见面、邮件、短信、视频等多种交互方式
- **参与者管理**：记录多人互动场景，关联其他联系人
- **时间追踪**：自动更新最后联系时间，支持"未联系天数"统计
- **历史查看**：完整的交互历史时间线展示

### 🔍 智能筛选与排序
- **多维筛选**：
  - 姓名关键词模糊匹配
  - 创建日期范围筛选
  - 未联系天数筛选（0-365天，5天间隔）
  - 标签筛选（OR逻辑）
- **排序方式**：
  - 按姓名排序
  - 按创建时间排序
  - 按最后联系时间排序
  - 按交互记录数量排序（待优化）

### 📊 统计功能
- **总数统计**：显示总联系人数
- **近期统计**：显示最近一个月内联系的人数
- **卡片视图**：在主页展示核心统计数据
- **实时更新**：统计数据实时同步更新

### 🌐 国际化支持
- **双语支持**：简体中文和英语
- **完整本地化**：界面文本、提示信息全面国际化
- **日期格式化**：支持本地化日期时间格式显示

## 操作指南

### 添加联系人
1. 点击「+」按钮进入联系人表单
2. 在「基本信息」标签页填写：
   - 姓名（必填）
   - 电话号码
   - 地址
   - 备注信息
3. 上传头像或选择图标及颜色
4. 添加标签（支持多个）
5. 添加自定义字段（键值对）
6. 切换到「记录」标签页查看交互历史
7. 点击保存完成创建

### 记录交互
1. 从联系人列表点击进入联系人详情
2. 在「记录」标签页点击「+」按钮
3. 选择交互日期和时间
4. 输入交互内容描述
5. 添加其他参与者（可选）
6. 保存记录（系统自动更新最后联系时间）

### 筛选和排序
1. 点击右上角筛选按钮打开筛选对话框
2. 设置筛选条件：
   - 姓名关键词
   - 创建日期范围
   - 未联系天数
   - 选中标签
3. 点击右上角排序按钮选择排序方式
4. 支持在列表视图和网格视图间切换

### 标签管理
1. 在联系人表单中点击「添加标签」
2. 输入标签名称（系统支持预定义颜色）
3. 已添加的标签以 Chip 形式显示
4. 点击标签可快速筛选相同标签的联系人

## 常见问题

**Q: 头像上传失败怎么办？**
A: 确保图片格式受支持（jpg, png等），大小不超过限制。

**Q: 如何批量添加标签？**
A: 目前需要逐个编辑联系人添加标签，建议统一规划标签名称。

**Q: 交互记录数量排序不准确？**
A: 当前实现存在已知问题，按交互记录数量排序功能待优化。

**Q: 可以导出联系人数据吗？**
A: 当前版本未实现导出功能，可通过文件管理器访问存储文件。

**Q: 删除联系人会丢失交互记录吗？**
A: 不会，系统会自动级联删除相关交互记录，保持数据一致性。

**Q: 标签颜色如何自定义？**
A: 系统提供预定义颜色方案（家人/朋友/同事/客户/重要），暂不支持完全自定义。
